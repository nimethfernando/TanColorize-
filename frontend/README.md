# Image Colorizer Frontend

React frontend for the Simple Colorizer application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

The app will open at `http://localhost:3000`

## Environment Variables

You can set the API URL by creating a `.env` file:
```
REACT_APP_API_URL=http://localhost:5000
```

## Usage

1. Make sure the Flask API backend is running (see main README)
2. Upload an image using the "Upload Image" button
3. Adjust the image size slider if needed
4. Click "Colorize Image" to process
5. Download the colorized result
